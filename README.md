# Go gRPC API Gateway - Learning Repo

This repo is for learning how to create API Gateway using Go and gRPC.

## Installation

Clone the repo.

```bash
git clone https://github.com/gaisuke/go-grpc-api-gateway.git
```

Run project
```bash
go run cmd/main.go
```

## Tech Stack
- Go
- Go-Validator
- Gin
